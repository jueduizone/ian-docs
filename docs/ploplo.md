# PloPlo - Web3 + AI 玩宠 NFT 项目官网 PRD

## 项目概述

**项目名称：** PloPlo  
**定位：** Web3 + AI 驱动的虚拟玩宠 NFT 平台  
**目标用户：** NFT 收藏家、宠物爱好者、Web3 游戏玩家、AI 技术爱好者  
**核心概念：** 结合 AI 个性化和区块链所有权的下一代电子宠物体验

## 设计风格参考

### 视觉风格
- **主色调：** 蓝紫色渐变（参考附件图片的半透明质感）
- **背景：** 深色背景 + 网格纹理（参考 magiclaw-poster.html）
- **质感：** 玻璃拟态（Glassmorphism）+ 霓虹发光效果
- **3D 元素：** 可交互的 3D 玩宠展示
- **动效：** 平滑过渡 + 微交互动画

### 设计趋势应用
1. **Web3 未来感** - 霓虹色 + 深空背景
2. **3D 沉浸式** - 可旋转/缩放的 3D 模型
3. **玻璃拟态** - 半透明卡片 + 背景模糊
4. **动态交互** - 鼠标悬停/点击反馈

## 技术架构

### 前端
- **框架：** Next.js 15 + TypeScript
- **3D 渲染：** Three.js / React Three Fiber
- **UI 库：** Tailwind CSS + Framer Motion
- **钱包集成：** Wagmi + Viem

### 智能合约
- **NFT 标准：** ERC-721
- **链：** Ethereum / Polygon（考虑 Gas 费）
- **功能：** 铸造、交易、属性存储

### AI 集成
- **性格生成：** 基于用户选择的 AI 模型
- **对话系统：** OpenAI / Claude API
- **行为模拟：** 规则引擎 + 机器学习

### 后端
- **数据库：** Supabase（用户数据、玩宠状态）
- **API：** Next.js API Routes
- **文件存储：** IPFS（NFT 元数据）

## 响应式设计

- **桌面端：** 完整 3D 体验 + 多列布局
- **平板端：** 简化 3D，保持核心功能
- **手机端：** 垂直布局，优化触摸交互
